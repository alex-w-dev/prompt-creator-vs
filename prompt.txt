- не должно быть отдельных кусков кода: файлы нужно вернуть полностью, даже если было минимальное изменение

- неизмененный код не должен быть пропущен: он должен вернуться в ответе
- неизмененный код нельзя заменять комментариями
- ответ не должен быть в виде diff формата
- возвращать всегда полный код
- возвращать только изменные файлы
- если в файле должны быть изменения, то содержимой файла ноужно возвращать целиком
- нельзя опускать код с пометкой "неизменнная часть кода"
- в конце ответа также выводить список файлов, которые были изменены
- в блоке кода нельзя первой строкой писать название файла, должен быть только код
- перед выдачей еще раз проверить код на валидность, при необходимости поправить
